5. TradingView Pine Script Alert Template
Add this Pine Script to any TradingView indicator or strategy to send automatic trade alerts to your journal. Replace the webhook URL with your journal's local URL.

PINE SCRIPT - Copy into TradingView Pine Editor
//@version=5 indicator("Maven Journal Alert", overlay=true)  // --- Configure your alert conditions below --- bullish_signal = ta.crossover(ta.ema(close, 9), ta.ema(close, 21)) bearish_signal = ta.crossunder(ta.ema(close, 9), ta.ema(close, 21))  // --- Alert message JSON --- buy_msg = '{"ticker": "' + syminfo.ticker + '", "action": "buy", "price": ' + str.tostring(close, "#.##") + ', "session": "NY Open", "setup": "OB+FVG", "timestamp": "' + str.tostring(timenow) + '"}'  sell_msg = '{"ticker": "' + syminfo.ticker + '", "action": "sell", "price": ' + str.tostring(close, "#.##") + ', "session": "NY Open", "setup": "OB+FVG", "timestamp": "' + str.tostring(timenow) + '"}'  // --- Create alerts --- alertcondition(bullish_signal, title="BUY Alert", message=buy_msg) alertcondition(bearish_signal, title="SELL Alert", message=sell_msg)  // --- In TradingView, set webhook URL to: http://localhost:3001/webhook/tradingview ---
